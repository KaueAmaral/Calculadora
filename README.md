# Calculadora
Trabalho da Calculadora - Professor Hugo Fumero

Esse projeto tem como objetivo o desenvolvimento de uma calculadora utilizando react native.

## Requisitos do Projeto:
- A calculadora devera conter os principais operadores aritméticos (+,-,*,/,=)
- devera ser possível apagar todo o calculo escrito de uma só vez pressionando AC
- para apagar apenas UM digito deverá utilizar a tecla C
